# FreshBite Cloud Kitchen - Cloud Kitchen Website

A modern, production-ready cloud kitchen ordering platform built with Next.js 15, React, TypeScript, Tailwind CSS, and Firebase.

## Quick Start

```bash
npm install
cp .env.example .env.local
# Add your Firebase and Razorpay credentials to .env.local
npm run dev
```

Visit http://localhost:3000

## Features

- Modern responsive design
- Dark/Light mode
- Firebase authentication
- Razorpay payments
- Shopping cart
- Admin dashboard
- SEO optimized
